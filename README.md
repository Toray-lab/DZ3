# Отчет о модульном домашнем задании N3
## Условия задания
Необходимо разработать программу строкового калькулятора, которая может обрабатывать:
+ операции: +, -, *, /;
+ круглые скобки ();
+ математические функции: sin, cos, tg, ctg, exp;
+ переменную x.
## Схема алгоритма
Алгоритм программы выполняется по следующим этапам:
+ Чтение введённого с клавиатуры выражения;
+ Разделение выражения на токены;
+ Перевод выражения в постфиксную форму;
+ Вычисление результата;
+ Вывод результата на экран;
## Вывод
В ходе выполнения модульного домашнего задания была создана программа удовлетворяющая условиям.
